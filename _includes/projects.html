Projects

  VFX Fractal Toolkit Set of tools for generating fractal and generative art. It was my graduation project, which I developed at Filmakademie Baden-Württemberg while studying Technical Directing. It contains tools written in OpenCL, OSL, Blink, Python, VEX and JavaScript intended to be used in Houdini, Arnold, Nuke or a web browser.
  VEX Reference Guide A guide I wrote for SideFX Houdini VEX scripting language. It tries to describe main language features and to provide examples of their usage. Can't remember the exact syntax for a feature in VEX? This guide will help you! :-). [blog post]
  Megascans to Houdini Integration Description of Megascans to Houdini integration which I was developing along with Peter Trandžík and Ondrej Poláček.
  Houdini 2 Open3D I built Houdini wrappers around Open3D functionality, which enable me to load point clouds, pre-process (downsample, estimate normals, compute FPFH feature) and perform Global and ICP registration on them to align them tightly. The aim of this project is to seamlessly integrate Open3D functionality with the Houdini toolset.
  Photogrammetry 1st, 2nd Part Documentation of photogrammetry workflow I developed for a student project. The first article describes photography workflow: acquisition, conversion, colors management and preparation for reconstruction software. The second article takes a look at the 3D reconstruction and geometry post processing.
  Houdini 2 VR Preview your Houdini stereo VR renders in a HMD device. This project links Houdini's Render View with web browser. It relies on WebVR and works even withouth a VR headset as a panorama viewer. [blog post]
  Showreel 04/2018 Compilation of VFX, animation shots and tools I've worked on. Check video description for breakdown.
  Filmakademie Showreel Showreel showing some of the projects I contributed to during my Technical Directing studies at Filmakademie Baden-Württemberg.
  Houdini RoyalRender Submitter WIP plugin for RoyalRender for submitting Houdini scenes to renderfarm. Implemented as a ROP node.
